== Description ==

This plugin lets you upload your own font files and apply them to any element of your website without requiring
a knowledge of html or css. All .otf and .ttf font files are supported.

Forget about using Flash and embed your fonts the right way.

Font Uploader embeds the fonts directly onto your web page using @font-face via CSS.

== Installation ==

1. Upload the 'font-uploader-free' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Font Uploader and configure your fonts

== Frequently Asked Questions ==

- Can I use this plugin to apply fonts to any html element, or just a select few? -

You can apply fonts to any element you wish using the "Advanced Custom Elements".

- Will my fonts work with Internet Explorer?

No, not with the free version, however the [$12 premium version](http://codecanyon.net/item/font-uploader/110175) has full support for IE fonts.

- What features are available in the premium version?

The $12 premium version has full support for Internet Explorer fonts, making your custom fonts %100 cross browser compatible for all desktop browsers.

The premium version also has support for Google Font Directory fonts.

Complete support for any and all issues is also offered with the purchase of the premium version.
